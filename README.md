## Quick Start

On Linux/macOS/Windows, use this line:

```bash
npm install -g meteor
```

Visit the official [install page](https://www.meteor.com/developers/install) to learn more.

Create a project:

```bash
meteor create my-app
```

Run it:

```bash
cd my-app
meteor
```

Or init existing project:


```bash
cd my-app
meteor npm install
```
Run it:

```bash
meteor
```
